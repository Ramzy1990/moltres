SideIntegralVariablePostprocessor

Computes an integral on internal sides of the specified variable

The InternalSideIntegralVariablePostprocessor is also an intermediate base class that should be derived from for any calculation involving the integral of a variable quantity over internal sides.

Example input syntax

In this example, an InternalSideIntegralVariablePostprocessor is used to compute the integral of variable u over all internal sides.

[Postprocessors]
  [./integral]
    type = InternalSideIntegralVariablePostprocessor
    variable = u
  [../]
[]
(moose/test/tests/postprocessors/internal_side_integral/internal_side_integral_test.i)
