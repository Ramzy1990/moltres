SideIntegralVariablePostprocessor

Computes a surface integral of the specified variable

The SideIntegralVariablePostprocessor is also an intermediate base class that should be derived from for any calculation involving the integral of a variable quantity over a side.

Example input syntax

In this example, a SideIntegralVariablePostprocessor is used to compute the intergral of variable u over the sideset of id 0.

[Postprocessors]
  [./integral]
    type = SideIntegralVariablePostprocessor
    boundary = 0
    variable = u
  [../]
[]
(moose/test/tests/postprocessors/side_integral/side_integral_test.i)

[Mesh]

  dim = 2
  nx = 10
  ny = 10
  xmin = 0
  xmax = 4
  ymin = 0
  ymax = 1
[]

  active = 'u'

  [./u]
    order = FIRST
    family = LAGRANGE
  [../]
[]

  active = 'diff'

  [./diff]
    type = Diffusion
    variable = u
  [../]
[]

  [./left]
    type = DirichletBC
    variable = u
    boundary = 3
    value = 0
  [../]

    type = DirichletBC
    variable = u
    boundary = 1
    value = 1
  [../]
[]

[]

    variable = u
  [../]
[]

[Outputs]
