ParsedNodeTransformGenerator

Applies a transform to a the x,y,z coordinates of a Mesh

Overview

The ParsedNodeTransformGenerator generator allows the user to specify parsed expressions to compute the new positions of all mesh nodes as the function of their old positions. The old node positions components are made available as function symbold x, y, and z. The new node position components are determined by the "x_function", "y_function", and "z_function" parmeters. The parsed expression syntax available is the same as used in ParsedFunction and ParsedMaterial.

Input Parameters

  • inputThe mesh we want to modify

    C++ Type:MeshGeneratorName

    Controllable:No

    Description:The mesh we want to modify

Required Parameters

  • constant_expressionsVector of values for the constants in constant_names (can be an FParser expression)

    C++ Type:std::vector<std::string>

    Controllable:No

    Description:Vector of values for the constants in constant_names (can be an FParser expression)

  • constant_namesVector of constants used in the parsed function

    C++ Type:std::vector<std::string>

    Controllable:No

    Description:Vector of constants used in the parsed function

  • show_infoFalseWhether or not to show mesh info after generating the mesh (bounding box, element types, sidesets, nodesets, subdomains, etc)

    Default:False

    C++ Type:bool

    Controllable:No

    Description:Whether or not to show mesh info after generating the mesh (bounding box, element types, sidesets, nodesets, subdomains, etc)

  • x_functionxFunction for the updated x component of the node

    Default:x

    C++ Type:ParsedFunctionExpression

    Controllable:No

    Description:Function for the updated x component of the node

  • y_functionyFunction for the updated y component of the node

    Default:y

    C++ Type:ParsedFunctionExpression

    Controllable:No

    Description:Function for the updated y component of the node

  • z_functionzFunction for the updated z component of the node

    Default:z

    C++ Type:ParsedFunctionExpression

    Controllable:No

    Description:Function for the updated z component of the node
