ADKernelValue

Description

This is an optimized Kernel base class for residuals which allow the test function (_test[_i][_qp]) to be factored out (see KernelValue).

(\dots,\psi_i)

The Jacobian in ADKernelValue is computed using forward automatic differentiation.
